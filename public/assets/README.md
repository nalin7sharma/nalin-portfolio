# Asset Placeholders

This folder contains editable SVG placeholders for the portfolio profile visual and project cards.

Recommended replacements:

- `profile-placeholder.svg`: replace with a professional portrait named `profile.webp`.
- `projects/*.svg`: replace with compressed WebP/AVIF screenshots or generated project visuals.
- Keep image dimensions stable so cards do not shift during loading.

The portfolio already references the current placeholder files, so the site works immediately after install.
