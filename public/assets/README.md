# Asset Placeholders

This folder contains optimized profile imagery and editable SVG project visuals.

Recommended replacements:

- `avatar/hero-ai-avatar.webp`: hero section reactive AI avatar.
- `avatar/about-pfp.webp`: About section profile picture.
- `projects/*.svg`: replace with compressed WebP/AVIF screenshots or generated project visuals.
- Keep image dimensions stable so cards do not shift during loading.

The portfolio already references the current optimized assets, so the site works immediately after install.
